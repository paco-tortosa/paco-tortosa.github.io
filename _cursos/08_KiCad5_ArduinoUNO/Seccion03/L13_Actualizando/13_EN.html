Update
  • In this lesson we are going to update the components that the KiCad template brought with the components of our new library.
  • Let's start with the schematic.
  • Let's start with P1, double click to access the properties.
  • As you can see there are no fields that we have created in our library. We have the option to update the fields with the red button and to change the reference to the library with the green button, but there is an easier way.
  • Cancel editing the properties and return to the schematic.
  • I am going to replace the P1 component with the Conn_01x08 component from our library.
  • With the shortcut key A add the component.
  • Filter by the name of the library and select the Conn_01x08 symbol.
  • Double click to edit its properties.
  • As reference set P2 and as field Value2Show the text Power
  • I am going to move the fields so that they are as in the template.
  • To move the reference, place the mouse over it and right-click to access the context menu to move it.
  • Do the same with the text Power
  • You can rotate the text with the R hot key while moving the component.
  • Select the component of the template and delete it with the delete key
  • Move the new component to the position of the template.
  • Do the same with components P2 to P8.
  • And it will be something like this:
  • Let's update the PCB.
  • There is an error because the components are locked on the PCB.
  • Select the Re-associate footprints by reference option and click the Update PCB button
  • Now there are no errors.
  • The placement of the components on the PCB has not changed.
  • Click on the P1 connector to see its properties.
  • As you can see, the reference is already to our footprint library.
