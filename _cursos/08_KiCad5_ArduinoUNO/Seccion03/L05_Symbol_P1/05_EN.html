P1 symbol
  • For the P1 and P4 components, we are going to copy the Conn_01x08 symbol from the Connector_Generic library in our library.
  • This is the connector that I found on Mouser. The part number is 5-535541-6.
  • It is 8 positions in one row and with pitch 2.54mm.
Steps
  • Open the symbol editor.
  • We look for it with the filter and save it in our library.
  • Save the library.
  • As we saw in the KiCad course, we are going to add the Value2Show, Mounted, MPN and < b> Manufacturer.
  • We are going to order the fields in the symbol, for this we set the grid to 10mils
  • We return the grid to 50mils which is the default grid of the schematic.
