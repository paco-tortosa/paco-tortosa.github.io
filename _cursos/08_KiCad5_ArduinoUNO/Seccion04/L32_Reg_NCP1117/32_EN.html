NCP1117 Regulator
  • The regulator U1 generates 5V from the input voltage at the Jack X1
  • This is the recommended regulator in the Arduino reference design. The manufacturer is ON Semiconductor and the part number is NCP1117ST50T3G
  • It can generate up to 1A and the output voltage is fixed (5V)
  • The maximum input voltage to the regulator is 20V
  • The minimum input voltage to the regulator will be the output voltage (5V) plus the dropout (around 1V), that is, about 6V .
  • But note that we have placed the D1 diode between the Jack and the regulator, and that D1 drops approximately 1V. So the minimum input on the Jack will be around 7V.
  • So a good power supply for the Jack would be between 9 and 12V.
  • In the library Regulator_Linear I found the symbol NCP1117-5.0_SOT223
  • I have copied it to the JAVIERM_LIBS library and added the usual fields in my designs.
  • In the footprint library Package_TO_SOT_SMD is the footprint SOT-223-3_TabPin2 which is exactly the footprint of the regulator so I have copied it into the library.
