Reset button
  • This is the button to manually reset the ATMEGA328P microcontroller
  • This is a push button that I found at Mouser
  • It is SMD and it has OFF-(ON) function, that is, when pressed it connects the contacts.
  • In the Switch library I found the SW_Push symbol.
  • I have copied it in the JAVIERM_LIBS library and added the usual fields in my designs.
  • In the Button_Switch_SMD library is the footprint of the component (SW_SPST_TL3305A).
  • So I have copied the footprint in our library.
