F1 Fuse
  • The F1 fuse limits 500mA, the current that the board can draw from the computer's USB port.
  • This is the fuse that I found at Mouser. The manufacturer is Bourns and its part number is MF-MSMF050-2.
  • In the Device library I found the Fuse_Small symbol.
  • And I have copied it in the JAVIERM_LIBS library.
  • This is the datasheet.
  • Here you can see the dimensions.
  • In the footprint library I have not found the exact footprint for this fuse so I have to take one and modify it.
  • In lesson 07 of the KiCad5 course we saw the traditional way of modifying a footprint, changing one by one all the dimensions of the elements of the footprint.
  • Here I am going to show you another way, directly hacking the footprint file. It is more difficult but very fast.
  • Copy the component into our library.
  • And give it the name of the component MF-MSMF050
  • Now we have the footprint in our library and we can modify it freely.
  • Open the file MF-MSMF050.kicad_mod with a text editor.
  • In red you can see the rectangle of the F.CrtYd layer, which is the one used to detect component overlaps on the PCB.
  • The rectangle is made up of 4 lines.
  • In green you can see the 2 pads with their position and size.
  • Run the program that I have made in C# KiCad_2Pads. You also have the source code of the program in case you want to modify it.
  • You have to specify the dimensions of the pad (wx and wy), the distance between them (d), and the distance C from the pad to the rectangle outline of the Courtyard layer which is usually 0.3mm
  • Enter the dimensions of the footprint that appear in the datasheet in the fields inside the rectangle in red and in the text field below the beginning and end of the sides of the rectangle are calculated in green and the position in orange and pad size.
  • Copy and paste in the footprint file.
  • And save the file.
  • If you double click on the footprint, the footprint is reloaded with the file data and you can see how our footprint looks.
  • Now we measure to verify that the dimensions are correct.
  • The distance between pads is correct (2.7mm)
  • Let's see the dimensions of the pad.
  • They are also correct.
  • This is another way to modify a footprint. Choose the one you like.
