Routing I
  • We are going to see 3 different ways to specify the width of tracks.
  • In this first lesson we are going to see the first way in which we choose the track width through the context menu by entering the width we want.
  • This option gives us a lot of freedom but also less control when the board is complex.
  • Open Pcbnew.
  • To route there is a hot key that is X.
  • And also a button.
  • We are going to start routing the 24V input, put the mouse on the pad and press the X key.
  • Right-click and select the menu item Use Custom Values ​​...
  • In the popup window enter 0.75mm
  • And we route the input 24V and the output 5V with width 0.75mm
  • We will now route GND with a width of 1mm.
  • The track between the led D2 and the resistor R1 is going to be of width 0.25mm
  • Let's see now how to unroute a track. First you have to select it with the mouse.
  • Only one section of the track has been selected.
  • With the right mouse button you can select the Single Track menu item from the context menu.
  • And with the delete key, unroute the track.
  • Let's route it again.
  • The last selected track width was 0.25mm, but we want the track width to be the same as the GND track.
  • There is an option to inherit the width of the track we started routing from. We activate it.
  • Now the track width is 1mm like the GND track.
  • If we wanted to unroute the entire GND track, we would do so with the Connected Tracks menu item.
  • And with the delete key:
  • We are going to route the GND track with a width of 0.5mm
  • We will finally see a way to unroute the entire board.
