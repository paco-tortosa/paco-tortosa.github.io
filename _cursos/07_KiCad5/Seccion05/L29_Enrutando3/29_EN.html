Routing III
  • In this lesson we will see the optimal way to route. We are going to use the Net classes.
  • Let's go to the schematic.
  • We are going to zoom in to see our components better.
  • The idea is to name the nets in the schematic.
  • Click on the Place net label tool.
  • Select the net of the output 5V and give it the name 5V.
  • Select the net of the 24V input and give it the name 24V.
  • Select the net of the 24V of the input connector and give it the name C24.
  • The same with the net between the led and the resistance.
  • Save the schematic.
  • Update the PCB.
  • Let's go to the configuration of the board.
  • In the Net Classes section add a new Net Class.
  • Give it the name Power and give it a track width of 0.75mm
  • Add another Net Class with a track width of 1mm and name Gnd
  • Now we have to assign the nets of the schematic to the Net Class of the PCB.
  • To the net 24V we assign the net class Power
  • We assign the net 5V and C24 the net class Power, the net Led the net class Default and to the net GND the net class Gnd.
  • And we save the PCB.
  • Now we can start routing, but first we make sure that the routing is going to take the width of the Net Class
  • Now we can route without worrying about the track width because it will be automatically selected depending on the net we are routing.
  • We have already routed our board, save the PCB before leaving.
