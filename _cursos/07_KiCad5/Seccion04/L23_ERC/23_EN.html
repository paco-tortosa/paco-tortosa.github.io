ERC
  • Now we have to see if our schematic contains errors. The tool is called ERC (Electrical Rules Checker)
  • As you can see there are two errors, Let'slet's solve the first one.
  • Pin 1 of the L7805 regulator is defined in the library as Power Input and is connected to the cathode of a diode that is not Power Output .
  • This error is very easy to solve, there is a symbol in the power library called PWR_FLAG which is a pin defined as Power Output . We are going to add it and connect it to pin1 of the regulator.
  • We are going to run the ERC tool again.
  • The first error is gone. The second is of the same type but with the GND symbol that we add.
  • We are going to duplicate the PWR_FLAG symbol and connect it to GND . Duplicating is done with the C hotkey.
  • We put the mouse on the symbol that we want to duplicate and press the C key.
  • We rotate the new component while moving it with the R key.
  • And finally we connect it to GND .
  • If we run ERC again we see that we no longer have errors.
  • Our schematic has no errors.
  • We can save it now.
