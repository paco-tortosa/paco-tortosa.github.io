Gerber
  • In order to manufacture our PCB we have to send the GERBER files and the drill files to the manufacturer.
  • Run the Plot tool
  • Select the subdirectory where to place the files. I'm going to call it Gerber.
  • Click on the Plot button to generate the files.
  • Now we have to generate the drill files (drills). Click the Generate Drill Files ... button
  • Select the subdirectory where to place the files (I'm going to put them in the same Gerber subdirectory).
  • Click on the Generate Drill File button to generate the files.
