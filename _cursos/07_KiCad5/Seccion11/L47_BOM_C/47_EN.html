BOM - C#
  • We are going to open with a text editor the schematic file (.sch).
  • Each component in the schematic has all the information delimited by a line that begins with $Comp and by another line that begins with $EndComp
  • The C# program reads the entire file and extracts all the component information.
  • You can download the source code and the executable for Windows from the download section of the lesson 1.
  • The program is called KiCad_BOM.exe, run it.
  • Click on the ... button to select the directory where the .sch files are
  • Click on the generate button.
  • The program has read all the .sch files in the directory and has generated the BOM in the BOM.csv file
  • We are going to run Excel.
  • We create a new file.
  • We are going to import the data from the csv file.
