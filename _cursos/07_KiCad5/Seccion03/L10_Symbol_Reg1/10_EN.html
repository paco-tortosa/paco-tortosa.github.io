Reg1 Symbol
  • We will see in this lesson how to create the symbol for the regulator Reg1.
  • This is the regulator I have chosen at Mouser. The part number is LM7805MPX / NOPB. It has a fixed output of 5V and an input of maximum 35V.
  • The package is SMD SOT-223-4 .
  • We are going to create a symbol for the regulator. We open the symbol editor and search for 7805 .
  • We are going to copy it into our library.
  • We will keep the name and select our library as a destination.
  • And save the library.
  • We are going to edit the properties of the symbol.
  • I am going to change the reference to Reg because I like this name better for regulators. I am going to add the Value2Show , Mounted , MPN and Manufacturer fields.
  • We are going to order the fields so that they do not overlap.
  • To do this, use a 10mils grid.
  • Finally save the library.
