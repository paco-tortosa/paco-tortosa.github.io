Símbolo de C1-C2
  • Vamos a ver en esta lección cómo crear el símbolo para los condensadores C1 y C2.
  • Esta es el condensador que he elegido en Mouser. El part number es 885012207098.
  • Es de 100nF y con encapsulado SMD de tamaño 0805.
  • Admite una tensión de 50Vdc que es mayor que la tensión que vamos a tener en el circuito que es de 24Vdc.
  • Vamos a crear un símbolo para el condensador. Abrimos el editor de símbolos y buscamos un condensador no polarizado.
  • Este es el símbolo que vamos a usar.
  • Vamos a copiarlo en nuestra librería.
  • Vamos a cambiar el nombre del símbolo, de C a C_100nF.
  • Seleccionamos nuestra librería (JAVIERM_LIBS) como destino.
  • Y guardamos.
  • Vamos a editar las propiedades del símbolo.
  • Vamos a añadir los campos Value2Show, Mounted, MPN y Manufacturer. Además en la descripción pondremos las principales características del condensador.
  • Vamos a ordenar el texto para que no se solape.
  • Para ello selecciona una rejilla de 10mils.
  • Por último guarda la librería.
