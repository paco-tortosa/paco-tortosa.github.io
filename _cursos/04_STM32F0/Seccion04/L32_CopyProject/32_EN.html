Copy a project
  • In this lesson I am going to copy the project from the previous lesson and I will modify it so that when the button B1 is pressed the led LD2 will switch on.
  • Eclipse is not a friendly environment to copy projects. That is why this lesson is about copying projects.
  • On this video you can see the program running.
Steps
  • Click on the New button and select the menu-item to create a new project from a .ioc file.
  • Browse to find the file .ioc.
  • Select the .ioc file from the project P14_HelloWorld.
  • Rename the project as P15_ButtonLed.
  • Now let's copy the source code files from the previous project.
  • Close all the projects but P15_ButtonLed.
  • Build and debug the project to search for errors.
  • Everything was fine. The creation of the project was correct.
  • Let's modify the file main.c to add the new funcionality.
