Blinking an led
  • In this lesson we will see a SW example from Renesas that blinks the led that the evaluation kit has on-board as you can see in the video.
  • Run e2 studio.
  • Specify where to create the workspace. In my case the folder workspace in the drive j.
  • Click on the below menu item to create a new Synergy C project.
  • There are several choices, select the first one (executable).
  • I am going to give the project the name Led.
  • Select the evaluation kit. By doing this the MCU is selected automatically and the pins used in the kit are configured also automatically.
  • Select the SW example named Blinky.
  • Once loaded the SW example, build the project.
  • As you can see there is no errors.
  • Debug the program.
  • Click on the button Resume to start the debug session.
  • The entry point to the program is the function main(). Be aware that the file main.c is auto-generated and you cannot modify it.
  • Click again on Resume to continue the debug session.
