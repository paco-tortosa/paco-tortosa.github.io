Led
  • In this first project let us blink an LED.
  • The CY8CKIT-042 kit will be used. It has an on-board RGB led.
  • It is called 'RGB led' because there are 3 leds in the same package. One Red, one Green and one Blue.
  • In this project you will use only the blue led, and you will blink it every half a second.
Steps
  • Execute PSoC Creator and create a new project.
  • Select the hardware that you will use (our CY8CKIT-042 kit).
  • Select empty schematic.
  • Enter where to save the project and workspace in your PC.
  • The workspace can contain many projects, that normally will have a relationship with each other.
  • Now you have created the project (but empty).
  • In the right pane there is a tab called Component Catalog.
  • Open it and select the Ports and Pins group.
  • The intention is to configure a pin as output and connect it to the blue led.
  • Select the option Digital Output Pin
  • Drag it to the schematic.
  • Double click on the pin to change its properties.
  • Rename it as you like (in my case Pin_Verde) and deselect HW connection to be able to control it from the program. Additionally select Initial drive state as high (1).
  • Let us explain it. If you watch the schematic of the kit, you will see how the leds (RGB) are connected. They share the same pin for the anode that is connected to P4_VDD (it can be 3.3V or 5V depending on the jumper J9, by default is connected to 3.3V)
  • To switch on the led a low level (0) is needed in the PSoC output pin. At reset you do not want the led to light, so the parameter Initial drive state must be 1.
  • Do the same with the Red and Blue leds. To do this select the new created pin in the schematic (Pin_Verde), copy it to paste it later on.
  • Double click on it to rename it and change its properties.
  • Do the same with the red led.
  • Now you have the 3 pins in the schematic.
  • But if you realize, it is still missing the pin where the led will be connected to.
  • To do this, double click on Proyecto01.cydwr.
  • And a tab is opened with the elements of the schematic on the right and the pins of PSoC on the left.
  • In the user guide of the kit you can see which pins are connected to the leds.
Led Pin
Red P1_6
Green P0_2
Blue P0_3
  • Drag the pins from the right panel to the corresponding pins in the left panel.
  • And so with all leds.
  • Build the project.
  • And check that there is no error (bottom panel)
  • What you have done so far, is simply define the pins, that are connected in the kit to the leds as digital output. Now it is time to program the activation of the blue led every half a second. Go to the file main.c.
  • Once open, you see that there is a zone reserved for your code.
  • You need to call a function called Pin_Azul_Write that will serve to activate/deactivate the digital output of PSoC. On starting to write the name of the function the autocomplete feature of PSoC Creator will give some suggestions.
  • It is needed also a function called CyDelay that will wait for an amount of time expressed in milliseconds. For half a second the value will be 500 (500 milliseconds = half a second).
20 forThe infinite loop will blink the blue led every half a second.(;;)
21 {
22 Pin_Azul_WriteBlue led off.(1);
23 CyDelayWait for half a second.(500);
24 Pin_Azul_WriteBlue led on.(0);
25 CyDelayWait for half a second.(500);
26 }
(hover the mouse over the highlighted words for information)
  • Build the project.
  • Now connect the kit to the computer (via USB).
  • And transfer the program.
  • The blue led blinks every half a second.
