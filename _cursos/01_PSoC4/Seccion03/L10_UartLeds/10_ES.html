UART y Leds
  • En este proyecto vas a usar el interfaz UART para encender los leds
  • PSoC enviará un mensaje al ordenador así :
  • Si envías un carácter entre 1 y 3 desde el ordenador, se encenderá un led según la siguiente tabla:
Carácter LED
1 rojo
2 verde
3 Azul
  • Conecta el kit como en el proyecto 009-Uart
Pasos
  • Copia el Workspace 1 y renómbralo como 10
  • Ve al esquemático
  • Busca el componente UART (SCB mode)
  • Arrastra el componente UART al esquemático y haz doble clic en él
  • Renómbralo como UART y selecciona 38400 como velocidad de transmisión
  • Genera la aplicación para generar la API (funciones) del nuevo componente
  • Asigna los pines UART del PSoC 4
  • Ve a "main.c" y borra el código antiguo:
  • Añade el siguiente código :
12 #include <project.h>

14 int main()

28 Pin_Verde_Write(1 Apaga el led verde);
