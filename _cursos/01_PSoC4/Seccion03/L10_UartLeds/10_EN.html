UART and LEDs
  • In this project you will use the UART interface to switch on the LEDs
  • PSoC will send a string to the PC as follows :
  • If you send a character from 1 to 3 from the PC, an LED will be switched on according to the following table:
Character LED
1 red
2 green
3 Blue
  • Connect the kit as in the 009-Uart project
Steps
  • Copy the Workspace 1 and rename it as 10
  • Go to the schematic
  • Search for the UART (SCB mode) component
  • Drag the UART component to the schematic and double click on it
  • Drag the UART component to the schematic and double click on it
  • Generate Application to generate the API (functions) of the new component
  • Assign the PSoC 4 UART pins
  • Go to "main.c" and delete the old code:
  • Add the following code :
12 #include <project.h>
