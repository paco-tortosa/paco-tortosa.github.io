PWM
  • En este proyecto vas a conocer un nuevo componente hardware, el componente PWM (Pulse Width Modulation)
  • Sirve para generar una señal que tiene una frecuencia fija y una parte variable llamada "Duty Cycle" que es el tiempo que está a 1
Pasos
  • Copia el Workspace 7 y renómbralo como Workspace 8
  • Ve al esquemático
  • Borra el contador
  • Ve al catálogo de componentes y arrastra el componente PWM al esquemático
  • Haz doble clic para acceder a la configuración
  • Renombra el componente como PWM
  • Modifica los siguientes valores :
Parámetro Descripción Valor
Period Periodo de la señal 10000
Compare Duración del Duty Cycle 5000
  • Así, la señal tendrá 5000 pulsos de reloj de duración de Duty Cycle (tiempo de la señal a 1) y 10000 pulsos de reloj de período
  • La señal de reloj es de 12 MHz
  • Habilita la conexión HW del led Rojo
  • Conecta el led Rojo con la salida "line" del componente PWM
  • Genera la aplicación
  • Ve a "main.c"
  • Borra el código del proyecto anterior
  • Añade la función "PWM_Start()" para incializar el componente PWM que añadimos al esquemático
  • Añade el siguiente código :
15 int main()
16 {
17 CyGlobalIntEnable; /* Enable global interrupts. */
18
19 PWM_Start();
20
21 for(;;)
22 {
23 PWM_WriteCompare( 500 ) Cambia el Duty Cycle a 500;
24 CyDelay(1000) Espera 1000 mseg = 1 segundo;
25 PWM_WriteCompare( 9500 ) Cambia el Duty Cycle a 9500;
26 CyDelay(1000) Espera 1000 mseg = 1 segundo;
27 }
28 }
  • Como siempre, compila el proyecto y transfiere el programa
