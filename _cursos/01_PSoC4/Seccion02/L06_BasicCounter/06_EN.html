Basic counter
  • In this project you will learn a new HW component, the Basic Counter.
  • As you can see in the picture, the Basic Counter has 3 inputs and 1 output.
Input Description
en enable (1) or disable (0) the counter.
reset when 1 resets the counter value to 0
clk If "en" = 1, a rising edge on this signal increments the counter value
  • You are going to use the Basic Counter to activate the Red and Green LEDs every time SW2 is pressed
Steps
  • Copy the Workspace 4 and rename it as Workspace 6
  • Go to the schematic
  • Delete the connection between Pin_SW2 and Pin_Rojo.
  • Remember that the Switch Button SW2 gives a 0 value when pressed. As you need a rising edge for the Basic Counter, a NOT component is needed to invert the signal
  • Go to the schematic and add it to the schematic
  • Now the Basic Counter
  • Connect Pin_SW2 to the input "clock" (through the NOT component)
  • Set the input "en" to 1 and the input "reset" to 0
  • Enable the "HW connection" for Pin_Verde
  • Connect the output of the Basic Counter to the Red and Green LEDs
  • Configure the Basic Counter as 2-bit width
  • Build the project
  • There are some errors
  • The output of the counter is a 2-bit bus, you must configure which bit goes to which led
