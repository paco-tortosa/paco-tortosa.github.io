Debouncer
  • In this project you will learn a new HW component, the Debouncer component
  • I have connected an external switch button to the pin P1[0] of PSoC
  • When the switch button is closed, on the digital input P1[0] there is ground, i.e. a logical 0
  • When the switch button is open, as the digital input P1[0] has an internal pull-up, there will be Vdd, i.e. a logical 1
  • I have chosen this external switch button because it is very "noisy"
  • In theory there would be a signal like (on the pin P1[0]) :
  • But there is a signal with bounces, both on the falling and rising edge:
  • The Debouncer component will remove the bounces and clean the signal
Steps
  • Create an empty project and name it as WS012
  • Create the following schematic :
  • In the component catalog, search for digital input and add it to the schematic
  • As you will use as external switch button rename it as "External SW"
  • Make sure to enable both "HW connection" and "External Terminal"
  • Additionally, the internal pull-up resistor must be enabled
  • Add the components "Ground" and "Switch"
  • "Ground" and "Switch" are "off-chip" components, used only for documentation
  • Rename SW_1 as SW
  • Connect Vss (Ground), SW and the "External_SW" pin
  • Add a digital output
  • And rename it as "Pin_Oscilloscope_Yellow". Make sure that "HW connection" is enabled
  • Copy it and paste it twice
  • Rename it the pasted pins as "Pin_Oscilloscope_Blue" and "Pin_Oscilloscope_Violet"
  • In the schematic you will have the digital input from the external switch button and 3 digital outputs
  • Connect the digital input to the digital output "Pin_Oscilloscope_Yellow"
  • Add the "Debouncer" component from the component catalog
  • Double click on it to edit its properties
  • Disable the generation of outputs on edge detection and rename the component as "Debouncer"
  • Connect the digital input to the input d of the Debouncer
  • Connect the output q of the Debouncer to the digital output Pin_Oscilloscope_Blue
  • A clock for the debouncer is needed
  • Edit the properties of the clock
  • Name it as ClockDebouncer and set the frequency to 50 Hz
  • Connect the clock to the digital output Pin_Oscilloscope_Violet
  • Build the project
  • There is an error on the digital output Pin_Oscilloscope_Violet. In PSoC4 to connect a clock signal to a digital output you have to do something else
  • Go to the properties of the digital output
  • Select Output mode as Clock
  • Select Output clocking as External
  • The digital output needs now a logical 1
  • It will be so:
  • Now you have to configure which pins are connected to the digital outputs and to the digital input
  • You do not need firmware for this project
  • Build the project and transfer the program
  • The following pictures are taken from the oscilloscope used for watching the digital outputs
  • In the following picture, you can see how the input signal (yellow) has bounces
  • The blue signal is already filtered, without bounces
  • The 50-Hz clock is active on the rising edge. Thatis why the filtered signal (blue) is synchronized with the rising edges of the clock signal
  • In the following picture, you can see how, if the input signal from the switch is shorter as the clock period (20 msec) there will be no signal generated
