Registro de Control
  • En este proyecto vamos a hacer parpadear el led rojo. Cada segundo vamos a cambiar automáticamente la frecuencia del parpadeo (de 5Hz a 20Hz). De esta manera veremos un parpadeo rápido y otro lento
  • En el proyecto anterior (015) el cambio lo hacíamos pulsando SW2. Ahora lo vamos a hacer desde el programa cada segundo, mediante el componente Control Register
Pasos
  • Copia el Workspace 15 y renómbralo como 16
  • Ve al esquemático
  • Borra el pulsador SW2
  • Añade un registro de control (Control Register)
  • Edita sus propiedades
  • Renómbralo a "Control_Register" y pon el número de salidas a 1
  • Añade el siguiente código en "main.c"
12 #include <project.h>
13
14 int main()
15 {
16 CyGlobalIntEnable; /* Enable global interrupts. */
17
18 for(;;)
19 {
20 Control_Register_Write(1) Pone a 1 la salida del registro de control. Este 1 selecciona la entrada 1 (20Hz) del multiplexor. El led parpadeará rápido;
21 CyDelay(1000) Espera durante 1 segundo;
22 Control_Register_Write(0) Pone a 0 la salida del registro de control. Este 0 selecciona la entrada 0 (5Hz) del multiplexor. El led parpadeará lento;
23 CyDelay(1000) Espera durante 1 segundo;
24 }
25 }
  • Como siempre, compila el proyecto y transfiere el programa
