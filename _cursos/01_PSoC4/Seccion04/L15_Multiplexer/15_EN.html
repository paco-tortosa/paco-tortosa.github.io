Multiplexer
  • In this project you will learn a new HW component, the Multiplexer component
  • It connects multiple input signals to one output signal, depending on the value of the control signal
  • In this project, you will change with the switch button SW2 which signal (5Hz or 20Hz) is supplied to the red LED
  • From a clock signal of 100 Hz, 2 signals ( 5Hz and 20Hz) are generated by 2 frequency dividers
  • The multiplexer operates as follows (let us supose a 2-bit multiplexer) :
  • When the control signal is 0, the output signal is connected to the input 0
  • When the control signal is 1, the output signal is connected to the input 1
  • When the control signal is 2, the output signal is connected to the input 2
  • When the control signal is 3, the output signal is connected to the input 3
  • In this project a 1-bit multiplexor is used. It has 2 inputs
Steps
  • Create an empty project and name it as WS015
  • Add a digital output
  • Rename it as Led_Red
  • Add a multiplexer
  • Rename it as "MUX" and configure the number of inputs to 2
  • Add a digital input
  • Rename it as SW2
  • Connect them
  • Add a frequency divider
  • Rename it as "FreqDiv_20" and set the divider value to 20
  • Connect a logical 1 to the enable input and a logical 0 to the reset input
  • Copy it
  • And paste it
  • Rename the new frequency divider as "FreqDiv_5" and set the divider value to 5
  • Connect both frequency dividers to the multiplexer
  • Add a clock
  • And configure it to 100 Hz
  • Connect it to both frequency dividers. And the schematic is done
  • Configure the pins of the PSoC
  • As always, build the project and transfer the program
