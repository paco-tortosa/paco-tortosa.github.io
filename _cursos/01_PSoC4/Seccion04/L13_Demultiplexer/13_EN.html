Demultiplexer
  • In this project you will learn a new HW component, the Demultiplexer component
  • It connects an input signal to multiple output signals, depending on the value of the control signal
  • If the control signal is 0, the input signal is connected to the output signal 0
  • If the control signal is 1, the input signal is connected to the output signal 1
  • If the control signal is 2, the input signal is connected to the output signal 2
  • If the control signal is 3, the input signal is connected to the output signal 3
  • The aim of this project is to change the led that is let, every time you push SW2
  • In the initial state the control signal is 0, that is why the output singal 0 of the demultiplexer is activated. The other output signals remain to 0, and as there is a NOT gate the digital output connected to the LED is 1 (remember that you need a 0 to light the LED)
  • On the first push of SW2 the counter value is incremented to 1 and the output signal 1 is activated, switching on the red LED
  • On the next push of SW2 the counter value is incremented to 2 and the output signal 2 is activated, switching on the green LED
  • On the next push of SW2 the counter value is incremented to 3 and the output signal 3 is activated, switching on the blue LED
  • On the next push of SW2 the counter value comes back to 0 and the output signal 0 is activated, no LED is on
Steps
  • Create an empty project and name it as WS013
  • Add a digital intput and 3 digital outputs
  • Rename the digital input as SW2
  • Rename the first digital output as Led_Red
  • Rename the second digital output as Led_Green
  • Rename the third digital output as Led_Blue
  • Add the demultiplexer
  • Add the NOT gates for the LEDs
  • Connect the outputs of the demultiplexer to the LEDs
  • Connect a logical 1 to the input of the demultiplexer
  • Add a basic counter
  • Rename it as "BasicCounter" and give it a width of 2 bits (it counts from 0 to 3)
  • Set the en (enable) signal to 1 and the reset signal to 0
  • Connect the switch button SW2 to the clock input through a NOT gate. Remember that every push of SW2 generates a negative pulse, and the basic counter needs a positive pulse
  • Connect the output of the basic counter to the control signal of the demultiplexer
  • Connect the pins of the schematic to the PSoC pins
  • As always, build the project and transfer the program
