iDAC
  • On this project you will use a potentiometer connected to the ADC component to switch on the red, green and blue LEDs
  • With the IDAC component you can generate a direct current that can be controlled with a 8-bit value
  • With a 2K resistor you can measure the current on the oscilloscope
  • From the pin P2[0] the current will flow through the resistor to ground
  • You can vary the current in a simple way to get a sawtooth signal
Steps
  • Create a new project
  • Based on the CY8CKIT-042 kit
  • With an empty schematic
  • And rename it as workspace 20
  • Go to the component catalog and search for the IDAC component
  • Add it to the schematic and edit its properties
Parameter Value Description
Polarity Positive (Source) The current goes out of PSoC
Resolution 8-bit The current will be controlled with a value from 0 to 255
Value 0 At startup the current is 0
Range 0-612 uA (2.4 uA/bit) The current is from 0 to 612 microamperes, i.e. 2.4 microamperes per bit
  • Add an output to the schematic
  • Edit its properties
  • Rename it as "Pin_IDAC" and select type "Analog"
  • Flip it
  • Connect IDAC and the output
  • Connect the output to the pin P2[0]
  • Add the following code to "main.c"
