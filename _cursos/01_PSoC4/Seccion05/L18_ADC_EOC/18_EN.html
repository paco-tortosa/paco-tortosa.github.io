ADC - EOC
  • On the previous project the ADC component was configured to read continously every 1 millisecond
  • Inside the function "main" an ADC value was read continously, that is why all read values during the same millisecond had the same value
  • There is an output signal of the ADC component to indicate that there is a new available measurement, it is the so-called "eoc" (end of conversion)
  • On this project the idea is to output this signal on an output pin and measure it with the oscilloscope
Steps
  • Copy the Workspace 17 and rename it as 18.
  • Add a digital output
  • Edit its properties
  • Rename it as "ADC_EOC"
  • Connect the output "eoc" of the "ADC" component with the output pin "ADC_EOC"
  • Assign the pin P0[2] to the digital output "ADC_EOC"
  • Connect an oscilloscope channel to the output pin P0[2]
  • On the oscilloscope you can see on the digital output a positive pulse every 1 millisecond
