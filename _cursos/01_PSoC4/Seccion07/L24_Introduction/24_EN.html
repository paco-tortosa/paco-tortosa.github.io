Introduction to Verilog
  • On this project you will use the programmable logic inside PSoC with Verilog, to create a component very simple.
  • It will have one input and two outputs. The output OUT1 is connected directly to the input IN1 and the output OUT1n is the negation of the input IN1.
  • In the next simulation you can see how it will work :
  • As you can see in the following picture, PSoC 4 has 4 UDB (Universal Digital Block).
  • Each UDB has 2 PLD (Programmable Logic Device).
  • A PLD has logical elements (AND, OR, INVERT and FLIP-FLOP).
  • You will program the UDB with Verilog that is a hardware description language used to model electronic systems.
  • In order to see the project running you must connect the input of the switch SW2 and the outputs of the leds Red and Green.
  • When SW2 is pressed the Red led will be on, when SW2 is not pressed the Green led will be on.
Steps
  • 2 projects are needed. The first is a library where the new component is implemented. The second one, main project, will use the new component.
Project 1 : Library
  • Create a new project.
  • Type = "Library project"
  • Set as project name LIB_024.
  • Go to the components tab.
  • Click on the right button of the mouse and select the option Add Component Item....
  • Name the component as PSoC4_Logic_01 and select Symbol Wizard.
  • Add 3 terminals. One input called in1 and 2 outputs called out1 and out1n.
  • Click with the right button of the mouse on an empty place of the schematics.
  • Click on Properties.
  • In the property Doc.CatalogPlacement click on the button with text ...
  • Add the text Community/Digital/Logic/PSoC4_Logic_01.
    PSoC4_Logic_01 is the name of the component.
    Community is the category in the component catalog.
    Digital and Logic are the subcategories.
  • Make the component wider so that the name fits in the box.
  • Click with the right button of the mouse on an empty place of the schematics.
  • And generate the Verilog file.
  • Click on the button Generate.
  • The Verilog file has been created with a module called like the component PSoC4_Logic_01.
  • Add the following lines of code:
17 module PSoC4_Logic_01 (
18 output out1,
19 output out1n,
20 input in1
21 );
22
23 //`#start body` -- edit after this line, do not edit this line
24
25 assign out1 = in1; Assign the input in1 to the output out1
26 assign out1n = ! in1 Assign the negation of the input in1 to the output out1n;
27
28 //`#end` -- edit above this line, do not edit this line
29 endmodule
Project 1 : Library
  • Create a new project.
  • Based on the CY8CKIT-042 kit.
  • Select empty schematic.
  • Name it P024
  • In the tab Components click on Dependencies...
  • Check the option LIB_024
  • Go to the schematic.
  • In the components catalog search for PSoC4_Logic_01
  • And copy it to the schematic.
  • Add a digital output to the schematic.
  • Edit its properties.
  • And rename it as Pin_LedRed
  • Copy and paste Pin_LedRed
  • And rename the new pin as Pin_LedGreen
  • Add a new digital input to the schematic.
  • Edit its properties, rename it as Pin_SW2 and select the mode Resistive pull up.
