Einführung: Editieren und Übersetzen
1 Um ein lauffähiges ejecutable C-Programm zu erzeugen producir, muss zunächst primero mit einem Texteditor eine
2 Quelltext-Datei angelegt ??? und mit Code gefüllt werden. Anschließend después wird ein Compiler
3 gestartet, der den Quellcode in Maschinen-Code übersetzt und ein lauffähiges Programm
4 erstellt crear, generar.
5 Als klassisches Beispiel soll hierzu para esto ein minimales Programm dienen servir, das lediglich sólamente "Hallo,
6 Welt!" auf dem Bildschirm pantalla ausgibt saca. Hierzu wird mit einem Texteditor folgender Code in
7 eine (neue) Datei hallo.c geschrieben:
1 // Datei: hallo.c /* 1. */
2 # include <stdio.h> /* 2. */
3 void main() /* 3. */
4 {
5 printf("Hallo, Welt!\n"); /* 4. */
6 }
1 Das obige de arriba Programm enthält folgende Komponenten:
2
3 1. Eine mit // eingeleitete Zeile línea am Dateianfang stellt einen Kommentar dar representa un comentario. Sie wird
4 beim Übersetzen durch den Compiler ignoriert und dient sirve lediglich der besseren Lesbarkeit a una mejor legibilidad.
5 Ebenso werden Textbereiche áreas de texto, die durch /* und */ begrenzt sind que están limitadas por /* y */, als Kommentare
6 für Erklärungen oder Hinweise genutzt
7
8 2. Mit der Anweisung instrucción #include <stdio.h> wird dem Compiler mitgeteilt mitteilen : comunicar, hacer saber, die
9 Standard-Input-Output-Bibliothek stdio.h zu laden. Diese von vielen C-Programmen
10 genutzte „Sammlung“ an Quellcode stellt u.a.unter anderen Funktionen für die Aus-
11 gabe von Text auf dem Bildschirm bereit.
12
13 3. Die Funktion main() startet das Hauptprogramm, das sich innerhalb der folgenden
14 geschweiften Klammern llaves befindet. Jedes C-Programm verfügt über tiene eine derartige tal, semejante, del estilo
15 main()-Funktion.
16
17 4. Durch den Aufruf der Mediante la llamada a Funktion printf() wird auf dem Bildschirm der in doppelten
18 Hochkommata Comillas dobles (literal : comillas de arriba) stehende Text ausgegeben. Die Zeichenfolge Secuencia de caracteres \n steht dabei ??? als
19 Zeichen für eine neue Zeile. Der Aufruf der Funktion muss, wie jede C-Anweisung,
20 mit einem Strichpunkt Punto y coma (;) ; beendet werden.
