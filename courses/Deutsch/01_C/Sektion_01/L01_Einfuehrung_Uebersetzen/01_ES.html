1 Um ein lauffähiges C-Programm zu erzeugen, muss zunächst mit einem Texteditor eine
2 Quelltext-Datei angelegt und mit Code gefüllt werden. Anschließend wird ein Compiler
3 gestartet, der den Quellcode in Maschinen-Code übersetzt und ein lauffähiges Programm
4 erstellt.
5 Als klassisches Beispiel soll hierzu ein minimales Programm dienen, das lediglich "Hallo,
6 Welt!" auf dem Bildschirm ausgibt. Hierzu wird mit einem Texteditor folgender Code in
7 eine (neue) Datei hallo.c geschrieben:
1 // Datei: hallo.c /* 1. */
2 # include <stdio.h> /* 2. */
3 void main() /* 3. */
4 {
5 printf("Hallo, Welt!\n"); /* 4. */
6 }
1 Das obige Programm enthält folgende Komponenten:
2
3 1. Eine mit // eingeleitete Zeile am Dateianfang stellt einen Kommentar dar. Sie wird
4 beim Übersetzen durch den Compiler ignoriert und dient lediglich der besseren Lesbarkeit.
5 Ebenso werden Textbereiche, die durch /* und */ begrenzt sind, als Kommentare
6 für Erklärungen oder Hinweise genutzt
7
8 2. Mit der Anweisung #include <stdio.h> wird dem Compiler mitgeteilt, die
9 Standard-Input-Output-Bibliothek stdio.h zu laden. Diese von vielen C-Programmen
10 genutzte „Sammlung“ an Quellcode stellt u.a. Funktionen für die Aus-
11 gabe von Text auf dem Bildschirm bereit.
12
13 3. Die Funktion main() startet das Hauptprogramm, das sich innerhalb der folgenden
14 geschweiften Klammern befindet. Jedes C-Programm verfügt über eine derartige
15 main()-Funktion.
16
17 4. Durch den Aufruf der Funktion printf() wird auf dem Bildschirm der in doppelten
18 Hochkommata stehende Text ausgegeben. Die Zeichenfolge \n steht dabei als
19 Zeichen für eine neue Zeile. Der Aufruf der Funktion muss, wie jede C-Anweisung,
20 mit einem Strichpunkt ; beendet werden.
