Control Register
  • In this project you will blink the red LED. Every second the blinking frequency will change ( from 5 Hz to 20 Hz). Thus you will see a fast and a slow blinking
  • In the previous project (015) the change was made by pushing SW2. Now it is going to be made automatically every second, by the use of the Control Register component
Steps
  • Copy the Workspace 15 and rename it as 16
  • Go to the schematic
  • Remove SW2
  • Add a Control Register component
  • Edit its properties
  • Rename it as "Control_Register" and set the output count to 1
  • Add the following code at "main.c"
12 #include <project.h>
13
14 int main()
15 {
16 CyGlobalIntEnable; /* Enable global interrupts. */
17
18 for(;;)
19 {
20 Control_Register_Write(1) Set the output of the Control Register to 1. This 1 selects the input 1 (20 Hz) of the multiplexer. The red LED blinks fast;
21 CyDelay(1000) Waits for 1 second;
22 Control_Register_Write(0) Set the output of the Control Register to 0. This 0 selects the input 0 (5 Hz) of the multiplexer. The red LED blinks slowly;
23 CyDelay(1000) Waits for 1 second;
24 }
25 }
  • As always, build the project and transfer the program
