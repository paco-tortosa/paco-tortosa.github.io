Contador
  • En este proyecto vas a conocer un nuevo componente hardware, el contador (Counter)
  • La secuencia que hay que programar es :
  • Cada 3 pulsaciones de SW2 se va a generar una interrupción, y en esa interrupción vas a conmutar el estado del led verde
  • Para hacer eso vas a usar un contador que vas a configurar con un valor inicial de 2. Cada vez que se pulse SW2 el contador será decrementado y cuando pase de 0 a 2 se generará la interrupción
Pasos

  • Ve al esquemático
  • Borra los componentes usados en el proyecto anterior
  • Como vas a controlar los leds (en este caso el verde) desde el programa tienes que deshabilitar la conexión HW
  • Añade un Timer Counter desde el catálogo de componentes
  • Se necesita un reloj para la entrada "clock"
  • Ahora hay que configurar el Timer Counter
  • Renómbralo como "Counter"
  • Modifica los siguientes valores :

Period 2 Interrupción cada 2 + 1 pulsaciones de SW2
