Counter
  • In this project you will learn a new HW component, the Counter
  • The sequence you have to program is :
  • Every 3 pushes of SW2 an interrupt will be generated and inside this interrupt you will toggle the green LED
  • To do this, you will use a counter, that must be configured with an initial value of 2. Every time you press SW2 the counter value will be decremented and when it passes from 0 to 2 an interrupt will be raised
Steps
  • Copy the Workspace 6 and rename it as Workspace 7
  • Go to the schematic
  • Remove the old stuff
  • As the leds (in this case the green one) will be controlled by the program, the "HW connection" must be disabled
  • Add a Timer Counter from the Component Catalog
  • A clock is necessary
  • Now it is time to configure the Timer Counter
  • Rename it as "Counter"
  • Modify the following values :
Parameter Value Description
Counter mode Down At every activation of the input "count" the counter value will be decremente
Input "count" Enabled and as falling edge The input will be active at the falling edge
Period 2 Interrupt every 2 + 1 pushes of SW2
  • Connect the switch button "Pin_SW2" to the input "count"
  • Add an interrupt component and connect it to the output "interrupt" of the Timer Counter
  • Rename the interrupt as "Counter_Int"
  • Generate the application
  • Go to "main.c"
  • Add the code :
