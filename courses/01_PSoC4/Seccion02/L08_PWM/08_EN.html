PWM
  • In this project you will learn a new HW component, the PWM component
  • You can generate a signal with a fixed frequency and a variable part called "Duty Cycle" that it is the time the signal is high
Steps
  • Copy the Workspace 7 and rename it as Workspace 8
  • Go to the schematic
  • Remove the counter
  • Go to the schematic and drag the PWM componente to the schematic
  • Double click to open the configuration window
  • Rename the component as PWM
  • Modify the following values :
Parameter Description Value
Period Signal period 10000
Compare Duty Cycle duration 5000
  • So, the signal will have 5000 clock pulses of Duty Cycle and 10000 clock pulses of Period
  • The clock signal is 12 Mhz
  • Enable the HW connection of the Red LED
  • Connect the Red LED with the "line" output of the PWM component
  • Generate the application
  • Go to "main.c"
  • Remove the previous project code
  • Add the function "PWM_Start()" to initialize the PWM component we added to the schematic
  • Add the following code :
15 int main()
16 {
17 CyGlobalIntEnable; /* Enable global interrupts. */
18
19 PWM_Start();
20
21 for(;;)
22 {
23 PWM_WriteCompare( 500 ) Change the Duty Cycle to 500;
24 CyDelay(1000) Wait for 1000 msec = 1 second;
25 PWM_WriteCompare( 9500 ) Change the Duty Cycle to 9500;
26 CyDelay(1000) Wait for 1000 msec = 1 second;
27 }
28 }
  • As always, build the project and transfer the program
