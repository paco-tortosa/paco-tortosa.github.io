Toggle Flip Flop
  • In this project you will learn a new HW component, the Toggle Flip Flop.
  • As you can see in the picture, the Toggle Flip Flop has 2 inputs and 1 output.
Input Operation
t enable (1) or disable (0) changes in the output pin q
clk If t = 1, a rising edge on this signal changes the output q
  • You are going to use the Toggle Flip Flop to toggle the Red LED every time SW2 is pressed.
Steps
  • Copy the Workspace 4 and rename it as Workspace 5.
  • Go to the schematic.
  • Delete the connection between Pin_SW2 and Pin_Rojo.
  • Remember that the Switch Button SW2 gives a 0 value when pressed. As you need a rising edge for the Toggle Flip Flop, a NOT component is needed to invert the signal.
  • Go to the schematic and add it to the schematic.
  • Now the Toggle Flip Flop.
  • Set the input t to 1.
  • And connect the output of the Toggle Flip Flop to the Pin_Rojo.
  • As always, build the project and transfer the program.
