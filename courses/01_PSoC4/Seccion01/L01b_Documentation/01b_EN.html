Improving the documentation
  • In the first project we added 3 pins to the schematic and we renamed them with meaningful names (Pin_Rojo, Pin_Verde and Pin_Azul), but this is not enough for someone who was not the author of the project to understand it.
  • Let us add some resistors and leds to the schematic.
Steps
  • First go to the schematic by clicking on b>TopDesign.cysch
  • Double click on Pin_Rojo
  • Enable the option External terminal so that the external terminal appears in the schematic.
  • In the Component Catalog search for resis and select the tab Off-Chip
  • Drag the resistor to the schematic.
  • Double click on the resistor and set the parameter 'Instance_Name_Visible' to false to remove the resistor name from the schematic.
  • Delete the value as well. So only the symbol of the resistor will be in the schematic.
  • Do the same to add an led.
  • The direction of the led has to be changed by using the button 'horizontal flip'
  • Do the same to add a power symbol.
  • Join the added components.
  • The schematic is done.
