Switch Button
  • We are going to use the swicth button (SW2) to control the RGB led.
  • You can find all the information about the CY8CKIT-042 kit in the user guide (go to the Downloads section of the lesson 01)
Steps
  • Go to the schematic and open the component catalog. Go to "Ports and Pins" and "Digital Input Pin".
  • Drag it to the schematic.
  • Double click on "Pin_1" to modify its properties.
  • In the user guide you can see that the name of the switch button is SW2 and it is connected to pin P0_7.
  • Modify the following properties :
Property Value
Name SW2
HW connection No
External terminal Yes
Drive Mode Resistive pull up
  • In the component catalog, select Switch.
  • Drag it to the schematic.
  • In the component catalog, select Ground.
  • Drag it to the schematic.
  • Select the Wire Tool to join the components.
  • Double click on the switch to rename it.
  • Double click on Proyecto02.cydwr to assign a pin to the switch.
  • Drag the Pin_SW2 and drop it to the pin P0_7.
  • Open main.c
  • Modify the source code to activate the blue led when the switch button SW2 is pressed.
  • The function Pin_SW2_Read() returns 0 when the switch button SW2 is pressed.
  • When the switch button SW2 is not closed, there is a logical 1 in the digital input P0_7 (due to the internal pull-up resistor), this logical 1 is copied to the digital output P0_3 and the LED is off because there is no voltage drop in its pins.
  • When the switch button SW2 is closed, there is a logical 0 in the digital input P0_7, this logical 0 is copied to the digital output P0_3 and the LED lights.
  • Build the project.
  • And transfer the program.
