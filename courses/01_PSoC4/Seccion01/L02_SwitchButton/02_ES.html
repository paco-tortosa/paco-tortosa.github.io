Pulsador
  • Vamos a usar el pulsador SW2 para encender el led RGB.
  • Puedes ver toda la información sobre el kit CY8CKIT-042 en la guía de usuario (ve a la sección de Descargas de la lección 01)
Pasos
  • Ve al esquemático y abre el catálogo de componentes. Busca dentro del grupo de "Ports and Pins" la entrada digital ("Digital Input Pin").
  • Arrástralo al esquemático.
  • Haz doble clic en "Pin_1" para editar sus propiedades.
  • En la guía de usuario vemos que el pulsador ("User Button") se llama SW2 y que está conectado al pin P0_7.
  • Modifica las siguientes propiedades :
Propiedad Valor
Nombre SW2
HW connection No
External terminal Sí
Drive Mode Resistive pull up
  • En el catálogo de componentes selecciona pulsador (Switch).
  • Arrástralo al esquemático.
  • En el catálogo de componentes selecciona masa (Ground).
  • Arrástrala al esquemático.
  • Selecciona la herramienta para unir componentes.
  • Haz doble clic para renombrar el pulsador.
  • Haz doble clic en Proyecto02.cydwr para asignar un pin del PSoC al pulsador.
  • Arrastra el Pin_SW2 desde la parte derecha al pin P0_7.
  • Abre main.c
  • Modifica el código para activar el led azul cuando pulses el pulsador SW2.
  • La función Pin_SW2_Read() devuelve 0 cuando el pulsador SW2 está pulsado.
  • Cuando el pulsador SW2 no está cerrado hay un 1 lógico en la entrada digital P0_7 (debido a la resistencia pull-up interna), este 1 se lleva a la salida digital P0_3 y el LED no se enciende porque no hay diferencia de potencial entre sus patas.
  • Cuando el pulsador SW2 está cerrado hay un 0 lógico en la entrada digital P0_7, este 0 se lleva a la salida digital P0_3 y el LED se enciende.
  • Compila el project.
  • Y transfiere el programa.
