New libraries
  • In this lesson I am going to create my symbol library for the schematic and my footprint library for the PCB.
  • I'm going to start with the symbol library. We go to the symbol editor.
  • In the File menu there is the menu item to create a new library (New Library ...)
  • I am going to call it JAVIERM_LIBS2 and I am going to put it in the libs subdirectory.
  • I will make it accessible only to this project.
  • If we filter by the word javierm we see that our library already appears.
  • Now we are going to create the footprint library. Let's go to the footprint editor.
  • In the File menu there is the option to create a new library (New Library ...)
  • I am going to call it JAVIERM_LIBF2 and I am going to place it in the libs subdirectory.
  • I will make it accessible only to this project.
  • If we filter by the word javierm we see that our library already appears.
