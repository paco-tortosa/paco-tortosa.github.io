L1 Ferrite
  • The L1 ferrite connects the ground of the USB connector with the chassis.
  • This is the recommended ferrite in the Arduino reference design. The manufacturer is Murata and the part number is BLM21PG600SN1D
  • In the library Device I found the symbol Ferrite_Bead_Small
  • I have copied it in the JAVIERM_LIBS library and added the usual fields in my designs.
  • These are the dimensions of the datasheet
  • In the KiCad libraries I have not found a footprint for a 0805 package ferrite, but the one for the capacitor works for us. Since as you can see the KiCad symbol is somewhat larger than what the datasheet recommends.
  • So I have copied the footprint to our library giving it the name Ferrite_0805
