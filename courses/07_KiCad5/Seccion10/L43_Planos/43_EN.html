Planes
  • In this lesson we are going to add 2 ground planes to the PCB, one in the TOP layer and the other in the BOTTOM layer.
  • We are going to select the copper layer in TOP (F.Cu)
  • We are going to use the add planes tool (Add filled zones)
  • Let's start at the upper left corner.
  • We have to specify the layer (F.Cu) and the net (GND) of the plane.
  • Draw the rectangle on the plane.
  • There are several options for displaying the planes.
  • With the first option the plane is filled with copper. You can see the plane very well but you cannot see the rest of the tracks.
  • With the second option, only the outline of the plane is shown.
  • And with the third option it shows the outline of the plane and the tracks of the TOP layer.
  • We are going to select only the outline view.
  • We are going to copy the TOP plane to generate the BOTTOM plane.
  • You must select a reference point for the copy.
  • And paste.
  • The reference point helps us to precisely locate the new plane.
  • We place the planes in the same position.
  • Now you have to edit the properties of the new plane. Since they are in the same position, KiCad asks us to indicate which plane we want to select.
  • The new plane has no associated Net.
  • Once selected we can edit the properties with the E hot key
  • We select the BOTTOM layer (B.Cu) and the net (GND)
  • We already have both planes.
  • Let's see how they look.
  • First the TOP layer
  • Now the BOTTOM layer
  • Let's see the PCB in 3D.
  • And we save the PCB.
  • Let's see if our board has errors.
  • It has no errors. We have already finished our board.
