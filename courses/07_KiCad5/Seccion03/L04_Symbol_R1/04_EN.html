R1 symbol
  • Here you can see the final schematic of the project. We are going to see in this lesson how to create the symbol for the resistor R1.
  • This is the resistor that I have chosen at Mouser. The part number is ESR10EZPJ102.
  • The resistance is 1K ohms, the power is 400mW, the tolerance is 5% and the SMT package is 0805.
  • We are going to create a symbol for the resistor R1. Open the Symbol Editor.
  • Instead of doing it from scratch, we are going to search for a symbol of any library of Kicad that could be suitable for the job.
  • In the library Device there is a resistor called R_Small that can fit.
  • We are going to create our own library.
  • The name of the library is JAVIERM_LIBS and is going to be in the folder libs.
  • We have to specify if the library is going to be available for all the projects (Global) or only for this project (Project). I am going to select Project.
  • I am going to copy the symbol R_Small of the library Device to our just create library (JAVIERM_LIBS).
  • I am going to change the name of the symbol, the new name will be R_1K
  • You have to specify to which library to copy. The below textbox is a filter, we write JAV and in the library list there is only our library JAVIERM_LIBS left.
  • Select our library.
  • And click on the Save button.
  • In order to find our new component, I use the library filter and write JAVIER.
  • And save the library.
  • Edit the symbol, click on the button Edit symbol properties.
  • Add a new field with a name Value2Show and value 1K. We will use it in the schematic to show the resistance value.
  • The name of the following field to add is Mounted and is to indicate if the component is going to be populated on the board or not.
  • The field MPN (Manufacturer Part Number) is the name that the manufacturer gives to the component.
  • The last field is Manufacturer and is the name of the manufacturer.
  • I only want to show in the schematic the fields Reference and Value2Show.
  • The alignment of the fields is going to be left and Bottom.
  • I am going to sort the fields in the symbol editor so they do not overlap.
  • For this I am going to set the grid to 10mils .
  • I am going to move some fields down.
  • I am going to focus on the ones that are going to be shown; Reference and Value2Show.
  • They are already perfect.
  • I am also going to put the Mounted field near the component.
  • I am done, I am going to put the grid back to 50mils .
  • Now I am going to add information related to the component in the description field.
  • Finally save the library.
