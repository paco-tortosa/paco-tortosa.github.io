Footprint de J1
  • Este es el datasheet del conector 1757019.
  • Vamos al editor de footprints para crear un footprint vacío para el conector.
  • Cogemos cualquier footprint, por ejemplo de una fiducial y lo copiamos a nuestra librería con el nombre null.
  • Borramos todo el footprint. Hay que seleccionar todo y borrar con la tecla suprimir.
  • Ordenamos un poco.
  • Y cambiamos el campo REF a la capa F.Fab que no vamos a usar en el PCB.
  • Guardamos el footprint.
  • Y asignamos el footprint null al símbolo.
