C1-C2 symbol
  • We will see in this lesson how to create the symbol for the C1 and C2 capacitors.
  • This is the capacitor that I have chosen at Mouser. The part number is 885012207098 .
  • It is 100nF and has a 0805 size SMD package.
  • It admits a voltage of 50Vdc that is greater than the voltage that we are going to have in the circuit, which is 24Vdc.
  • We are going to create a symbol for the capacitor. We open the symbol editor and look for a non-polarized capacitor.
  • This is the symbol that we are going to use.
  • Let's copy it in our library.
  • We are going to change the name of the symbol, from C to C_100nF .
  • We select our library ( JAVIERM_LIBS ) as destination.
  • And we save.
  • We are going to edit the properties of the symbol.
  • We are going to add the Value2Show , Mounted , MPN and Manufacturer fields. Also in the description we will put the main characteristics of the capacitor.
  • We are going to order the text so that it does not overlap.
  • To do this, select a grid of 10mils .
  • Finally save the library.
