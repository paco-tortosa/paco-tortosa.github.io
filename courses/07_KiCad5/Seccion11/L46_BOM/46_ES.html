BOM
  • Para obtener la lista de materiales (BOM) vamos al editor de esquemáticos.
  • En el menú Tools está la opción de generar la lista de materiales (Generate Bill of Materials...)
  • Hay varios plugins para obtener distintas modalidades de BOM.
  • Vamos a generar el BOM con el plugin boom_csv_grouped_by_value
  • Se ha generado un fichero xml. Lo abrimos con excel.
  • Sale un BOM un poco raro. En la siguiente lección os voy a mostrar un programa en C# para obtener un BOM en formato csv.
