PWM (HAL)
  • In this lesson we are going to do the same as in the previous lesson (generate a PWM signal of period 100us and duty cycle 25%) but this time using the HAL and the configurator MX.
Steps
  • Create a new project based on the evaluation kit with the name P22_TIM1_PWM_HAL.
  • Let's configure the TIM1 with the same parameters than in the previous lesson.
  • The clock for TIM1 will be the MCU internal clock (48MHZ).
  • Configure the channel 1 as PWM output.
  • The prescaler must be 47 to get a frequency of 1MHz from the input clock of 48MHz, the counter mode as upcounting and the register ARR 99 that makes the period be 99+1=100 us.
  • Note that the pin PA8 has been configured as TIM1_CH1.
  • Set the auto-reload preload to Enable.
  • Now is time to set the duty cycle to 25us.
  • Now you can generate the source code for the configuration.
  • Go to the file main.c
  • Let's add here the activation of TIM1.
  • With the function HAL_TIM_PWM_START the TIM1 is activated and the PWM signal is generated on the pin PA8.
  • You can build and debug the program.
